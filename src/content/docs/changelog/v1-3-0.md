# v1.3.x Changelog

This version series focuses on custom domain support, enhanced local proxy reporting, and initial daily telemetry SQLite bucketing.

---

## v1.3.1 — Proxy Status & Custom Domains

### What Changed

* **Custom API Domains**: Users can now configure a custom API endpoint domain using environment variables or configuration overrides.
* **Telemetry Email Association**: Sync logs now optionally associate telemetry events with logged-in user emails for easier workspace tracking.
* **Proxy Status Enhancements**: The `agentsecrets proxy status` command has been enhanced to return more detailed info regarding background socket connectivity.

---

## v1.3.0 — Telemetry & Snapshots

### What Changed

* **Daily Telemetry Bucketing**: Re-designed telemetry logs to bundle events into daily local SQLite database records instead of single-event writes.
* **Context-Aware Snapshots**: Captured environment configurations to detect secret drift between local developer machines and cloud state.
